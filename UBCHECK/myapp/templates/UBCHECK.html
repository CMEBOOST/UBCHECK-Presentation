UB

CHECK

UBCHECK คือแอปพลิเคชันบน Android ที่พัฒนาขึ้นเพื่อช่วยป้องกันและรับมือกับภัยคุกคามจาก Scammer ในชีวิตประจำวัน ครอบคลุมตั้งแต่การโทรศัพท์รบกวน การส่ง SMS และลิงก์อันตราย ไปจนถึงการใช้เสียงสังเคราะห์จาก AI เพื่อหลอกลวง

เมื่อมีสายโทรเข้าหรือ SMS ที่น่าสงสัย UBCHECK จะทำการตรวจสอบและแจ้งเตือนผู้ใช้โดยอัตโนมัติ พร้อมบล็อกเบอร์หรือข้อความที่ตรวจพบว่าเป็นอันตราย นอกจากนี้ยังสามารถสแกน QR Code และตรวจสอบ URL ได้ทันที เพื่อป้องกันไม่ให้ผู้ใช้ตกเป็นเหยื่อของเว็บไซต์ฟิชชิ่ง

สำหรับปัญหาเสียงสังเคราะห์ AI ที่มิจฉาชีพนำมาใช้แอบอ้างตัวตน UBCHECK มีระบบตรวจจับ Deepfake Voice ที่ช่วยวิเคราะห์ว่าเสียงที่ได้ยินเป็นเสียงมนุษย์จริงหรือเสียงที่สร้างขึ้นด้วย AI

นอกเหนือจากการป้องกัน UBCHECK ยังมี AI Assistant ที่พร้อมให้คำปรึกษาเบื้องต้นแก่ผู้ใช้ที่ไม่แน่ใจว่ากำลังถูกหลอกอยู่หรือเปล่า หรือสำหรับผู้ที่ตกเป็นเหยื่อแล้วและไม่รู้ว่าต้องทำอะไรต่อไป ระบบจะแนะนำขั้นตอนการรับมือตั้งแต่การอายัดบัญชีไปจนถึงการแจ้งความ เพื่อลดความเครียดและลดอุปสรรคสำหรับผู้ที่ไม่กล้าขอความช่วยเหลือ

ทั้งนี้ UBCHECK ยังมีระบบติดตามข่าวสารและสถิติ scam ที่กำลัง trending ในไทยแบบ real-time เพื่อให้ผู้ใช้รับรู้รูปแบบกลโกงใหม่ๆ อยู่เสมอ และสามารถระวังตัวได้ก่อนที่จะตกเป็นเหยื่อ

ฟังก์ชันที่คาดว่าจะมีของ UBCHECK

Call Screening + Community Alert

คัดกรองเบอร์โทรขาเข้าแบบ real-time เทียบกับฐานข้อมูล scam และกระจายแจ้งเตือนชุมชนทันทีเมื่อมี report ใหม่ คล้าย Waze

Must have
  • มีสายเข้า Android OS ตรวจจับสายเข้าผ่าน CallScreeningService — แอปรับ event โดยอัตโนมัติโดยไม่ต้องเปิดแอปค้างไว้
  • ส่งเบอร์ไปตรวจสอบ Backend Flutter ส่งเบอร์โทรศัพท์ไปยัง FastAPI endpoint ผ่าน HTTPS แบบ real-time ก่อนผู้ใช้รับสาย
  • เช็ค Phone DB + Blacklist Backend ค้นหาเบอร์ใน PostgreSQL (ฐานข้อมูล scam) และ Redis (cache สำหรับความเร็ว) พร้อมเช็ค blacklist จากแหล่งภายนอก
  • แสดงผลบนหน้าจอ แอปแสดง label ทับบนหน้าสาย — 🔴 เสี่ยงสูง / 🟡 น่าสงสัย / 🟢 ปลอดภัย / ⚪ ไม่รู้จัก พร้อมจำนวนครั้งที่ถูก report
  • ผู้ใช้ Report เบอร์ใหม่ หลังวางสาย ผู้ใช้กด report พร้อมระบุประเภท เช่น Call center, สินเชื่อ, แก๊งคอลเซ็นเตอร์ — ข้อมูลบันทึกลง PostgreSQL
  • Community Alert กระจายแจ้งเตือน เมื่อเบอร์ถูก report ครบ threshold ที่กำหนด Backend broadcast ผ่าน WebSocket แจ้งเตือนผู้ใช้ทุกคนที่ online อยู่ทันที คล้าย Waze
