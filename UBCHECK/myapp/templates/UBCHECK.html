UB

CHECK

UBCHECK คือแอปพลิเคชันบน Android ที่พัฒนาขึ้นเพื่อช่วยป้องกันและรับมือกับภัยคุกคามจาก Scammer ในชีวิตประจำวัน ครอบคลุมตั้งแต่การโทรศัพท์รบกวน การส่ง SMS และลิงก์อันตราย ไปจนถึงการใช้เสียงสังเคราะห์จาก AI เพื่อหลอกลวง

เมื่อมีสายโทรเข้าหรือ SMS ที่น่าสงสัย UBCHECK จะทำการตรวจสอบและแจ้งเตือนผู้ใช้โดยอัตโนมัติ พร้อมบล็อกเบอร์หรือข้อความที่ตรวจพบว่าเป็นอันตราย นอกจากนี้ยังสามารถสแกน QR Code และตรวจสอบ URL ได้ทันที เพื่อป้องกันไม่ให้ผู้ใช้ตกเป็นเหยื่อของเว็บไซต์ฟิชชิ่ง

สำหรับปัญหาเสียงสังเคราะห์ AI ที่มิจฉาชีพนำมาใช้แอบอ้างตัวตน UBCHECK มีระบบตรวจจับ Deepfake Voice ที่ช่วยวิเคราะห์ว่าเสียงที่ได้ยินเป็นเสียงมนุษย์จริงหรือเสียงที่สร้างขึ้นด้วย AI

นอกเหนือจากการป้องกัน UBCHECK ยังมี AI Assistant ที่พร้อมให้คำปรึกษาเบื้องต้นแก่ผู้ใช้ที่ไม่แน่ใจว่ากำลังถูกหลอกอยู่หรือเปล่า หรือสำหรับผู้ที่ตกเป็นเหยื่อแล้วและไม่รู้ว่าต้องทำอะไรต่อไป ระบบจะแนะนำขั้นตอนการรับมือตั้งแต่การอายัดบัญชีไปจนถึงการแจ้งความ เพื่อลดความเครียดและลดอุปสรรคสำหรับผู้ที่ไม่กล้าขอความช่วยเหลือ

ทั้งนี้ UBCHECK ยังมีระบบติดตามข่าวสารและสถิติ scam ที่กำลัง trending ในไทยแบบ real-time เพื่อให้ผู้ใช้รับรู้รูปแบบกลโกงใหม่ๆ อยู่เสมอ และสามารถระวังตัวได้ก่อนที่จะตกเป็นเหยื่อ

ฟังก์ชันที่คาดว่าจะมีของ UBCHECK

Call Screening + Community Alert

คัดกรองเบอร์โทรขาเข้าแบบ real-time และกระจายแจ้งเตือนชุมชน

Must have
  • มีสายเข้า Android OS ตรวจจับสายเข้าผ่าน CallScreeningService — แอปรับ event โดยอัตโนมัติ
  • ส่งเบอร์ไปตรวจสอบ Backend Flutter ส่งเบอร์โทรศัพท์ไปยัง FastAPI endpoint ผ่าน HTTPS
  • เช็ค Phone DB + Blacklist Backend ค้นหาเบอร์ใน PostgreSQL และ Redis
  • แสดงผลบนหน้าจอ แอปแสดง label — 🔴 เสี่ยงสูง / 🟡 น่าสงสัย / 🟢 ปลอดภัย
  • ผู้ใช้ Report เบอร์ใหม่ หลังวางสาย ผู้ใช้กด report บันทึกลง PostgreSQL
  • Community Alert กระจายแจ้งเตือนผ่าน WebSocket เมื่อถึง threshold ที่กำหนด

SMS + Link + QR Scanner + Screen shot

อ่าน SMS อัตโนมัติ, ตรวจ phishing URL และสแกน QR Code

Must have
  • รับข้อมูล 3 ช่องทาง: READ_SMS (Android), Paste URL เอง และสแกนจากกล้อง
  • Decode QR / Extract URL: mobile_scanner และ NLP สำหรับดึงลิงก์จากข้อความ
  • วิเคราะห์ URL: PhishTank, Whois API (domain age), SSL Check และ NLP pattern
  • คำนวณ Risk Score: 0–100 พร้อมระบุ Scam Type (Fake bank, Investment fraud ฯลฯ)
  • แสดงผลและแจ้งเตือน: แสดงผลลัพธ์พร้อมเหตุผลความเสี่ยง และมีปุ่ม Report

History + Trending Dashboard

ประวัติการตรวจสอบส่วนตัว + สถิติ scam ที่กำลัง trending ในไทยแบบ real-time พร้อมกราฟและ Top 10 เบอร์อันตราย

Must have
  • บันทึก Log ทุก action ทุกครั้งที่ผู้ใช้ตรวจสอบ (เบอร์, SMS, QR, เสียง) ระบบบันทึก log ลง PostgreSQL พร้อม timestamp, ผลลัพธ์ และ Risk Score
  • หน้า My History ผู้ใช้เห็นประวัติการตรวจสอบทั้งหมดของตัวเอง กรองตามประเภท (เบอร์ / SMS / QR / เสียง) และวันที่ กดดู detail แต่ละรายการได้
  • Trending Engine รวบรวม Community data Backend รวบรวม report ทั้งหมดจากผู้ใช้ทุกคน คำนวณ stats เช่น เบอร์ที่ถูก report มากที่สุด, ประเภท scam ที่กำลัง trending, domain อันตรายใหม่ที่เพิ่งถูกพบ
  • หน้า Trending Dashboard แสดงกราฟและ Top list แบบ real-time เช่น Top 10 เบอร์อันตราย, สถิติ scam แยกตามประเภทและภูมิภาค, กราฟ trend รายสัปดาห์/เดือน

Auth — ระบบสมาชิก

Login / Register + JWT Token + bcrypt + Rate limiting ป้องกัน spam report + ข้อมูลเข้ารหัสทั้งหมด

Must have
  • สมัครสมาชิก ผู้ใช้กรอก email + password — Backend hash รหัสผ่านด้วย bcrypt ก่อนบันทึกลง PostgreSQL ไม่เก็บ plain text เด็ดขาด
  • เข้าสู่ระบบ + รับ JWT Token เมื่อ login สำเร็จ Backend ส่ง JWT Access Token (อายุ 1 ชั่วโมง) + Refresh Token (อายุ 30 วัน) กลับมา — Flutter เก็บ token ใน Secure Storage
  • ทุก Request แนบ JWT ทุก API call แนบ Authorization: Bearer token ใน header — Backend ตรวจสอบ token ก่อนประมวลผลทุกครั้ง
  • Rate Limiting + Security จำกัด report ได้สูงสุด 10 ครั้ง/ชั่วโมง/user เพื่อป้องกัน spam — ข้อมูลประวัติการตรวจสอบเข้ารหัสใน database

Post-call AI Voice Analyzer

อัปโหลดเสียงหลังวางสาย — Whisper แปลงเป็นข้อความ, LCNN ตรวจจับ Deepfake Voice, Claude วิเคราะห์ scam pattern

Should have
  • อัปโหลดไฟล์เสียง ผู้ใช้อัปโหลดไฟล์เสียงจากการโทรที่น่าสงสัย รองรับ .mp3, .wav, .m4a — หรือบันทึกเสียงผ่านกล้อง speaker โดยตรงบนแอป
  • Whisper — Speech-to-Text Whisper API แปลงเสียงเป็นข้อความภาษาไทย รองรับสำเนียงและภาษาพูดทั่วไป ได้ transcript ที่ค่อนข้างแม่นยำ
  • LCNN — Deepfake Detection โมเดล LCNN (Light CNN) วิเคราะห์ waveform เสียงโดยตรง ตรวจหา artifact ที่เสียงสังเคราะห์จาก AI มักทิ้งไว้ เช่น spectral inconsistency, unnatural prosody — train บน ASVspoof 2021 dataset
  • Claude API — Pattern Analysisส่ง transcript ให้ Claude วิเคราะห์ว่ามี scam pattern หรือไม่ เช่น เร่งโอนเงิน ("โอนภายใน 30 นาที"), แอบอ้างหน่วยงาน ("ผมเป็นตำรวจ DSI"), สร้างความกลัว ("บัญชีคุณถูกอายัด"), สร้างความโลภ ("ได้รับรางวัล")
  • รวมผลและแสดง Report แสดงผลรวม: Deepfake Score (จาก LCNN) + Scam Pattern Score (จาก Claude) + Transcript ข้อความ + highlight ส่วนที่น่าสงสัย + คำแนะนำว่าควรทำอะไรต่อ

AI Chat Assistant

"โดนหลอกทำยังไง?" — Claude แนะนำขั้นตอน step-by-step ตั้งแต่อายัดบัญชีจนถึงแจ้ง ปปง. พร้อม quick action กดโทรหาธนาคารได้เลย

Should have
  • ผู้ใช้เปิดหน้า Chat ผู้ใช้พิมพ์อธิบายสถานการณ์ที่เจออยู่ เช่น "มีคนโทรบอกว่าได้รับรางวัล ให้โอนค่าธรรมเนียมก่อน" หรือ "โอนเงินไปแล้ว ทำยังไงดี"
  • ส่งข้อความไปยัง Claude API Backend ส่ง system prompt ที่มี context เกี่ยวกับ scam ไทย + ข้อมูลหน่วยงานที่เกี่ยวข้อง + เบอร์ด่วนธนาคารไทย รวมกับคำถามของผู้ใช้ไปยัง Claude API
  • Claude วิเคราะห์และตอบ Claude ระบุว่าเป็น scam pattern ประเภทไหน และให้คำแนะนำ step-by-step เช่น ถ้าโอนเงินไปแล้ว: (1) โทรธนาคารขอ อายัดบัญชีปลายทาง (2) แจ้งความออนไลน์ที่ thaipoliceonline.com (3) แจ้ง ปปง. ถ้าเงินเกิน 50,000 บาท (4) เก็บ screenshot ทุกอย่างไว้เป็นหลักฐาน
  • แสดงผลพร้อม Quick Action แสดงคำตอบพร้อมปุ่ม quick action เช่น ปุ่มโทรหาธนาคาร, ปุ่มเปิดเว็บแจ้งความ, ปุ่มบันทึก chat เป็น PDF เพื่อเอาไปแจ้งความ

Background Service

Android Foreground Service monitor สายเข้าและ SMS ตลอดเวลาโดยไม่ต้องเปิดแอปค้างไว้ แจ้งเตือน notification ทันที

Nice to have
  • เปิด Background Service ผู้ใช้เปิดใช้งาน Background Protection — Android สร้าง Foreground Service ที่แสดง notification ถาวรว่า UBCHECK กำลังป้องกันอยู่
  • Monitor สายเข้า Service รับ event สายเข้าจาก Android OS แม้ปิดแอปอยู่ — ตรวจสอบเบอร์และแสดง notification ทันทีว่าเบอร์นี้เสี่ยงหรือไม่
  • Monitor SMS ขาเข้า รับ SMS ใหม่ผ่าน BroadcastReceiver — วิเคราะห์ URL ใน SMS อัตโนมัติ ถ้าพบ phishing link แจ้งเตือน notification ทันที

Deepfake Voice Detection

QR Code Phishing Scanner
